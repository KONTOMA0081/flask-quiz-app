# Mini Quiz Flask App

This project is a simple quiz web application built using Python and Flask.

Users can answer questions through a web page, and the system will calculate their score and show the result with a timestamp.

Features
- Quiz questions
- Score tracking
- Timestamp for results
- Queue data structure
- Object-Oriented Programming

How to Run the App

1. Install Python
2. Install Flask by running:

pip install flask

3. Run the application:

python app.py

4. Open a browser and go to:

http://127.0.0.1:5000


The quiz will appear and you can answer the questions.
